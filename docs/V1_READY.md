# Context Fabric v1.0

Context Fabric v1.0 is ready for public SDK evaluation and private core deployment.

## Included

- Public preflight SDK.
- Generic channel router.
- Budget profiles.
- Policy audit helpers.
- Repo context packs.
- Trusted-backend API client.
- Sanitized fixtures and templates.

## Not included

- Private core internals.
- Operator channel maps.
- Secrets, credentials, or raw memory exports.
